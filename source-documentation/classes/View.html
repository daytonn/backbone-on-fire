  • [options] Object optional

    There are several special options that, if passed, will be attached directly to the view: model, collection, el, id, className, tagName, attributes, renderLoadingMask and events. If the view defines an initialize function, it will be called when the view is first created. If you'd like to create a view that references an element already in the DOM, pass in the element as an option: new View({el: existingElement})
